# Project Management Console App

A console-based project management application written in **Go**.

## Overview
This project manages employee data, project tasks, deadlines, budgets, and task status through a menu-driven terminal interface.

## Features
- Employee data management
- Task and project data entry
- Search by employee name, project, and jobdesk
- Edit and delete project-related data
- Sort tasks by priority, date, status, and deadline
- Budget total calculation

## Tech Stack
- Go

## Run Locally
```bash
go run mainMp.go
```
